# Remotion Runtime Pack

## Use When

- The video should be maintainable React code.
- The project needs reusable templates, data visualizations, or parameterized props.
- Stability is more important than maximal visual experimentation.

## Programmer Rules

- Scaffold a Remotion project when no project exists.
- Build compositions as React components.
- Keep captions, assets, audio, and scene timing parameterized.
- Run a still-frame check before final render when practical.

## Acceptance Checks

- Source project builds.
- Composition duration and dimensions match `runtime_plan`.
- Captions do not overflow.
- Final package includes source and render command.
